Autogro Pi configuration 5-9-23
_________________________

SPI on Pi must be turned on for MCP AtoD to work, use (sudo raspi-config)

Turn on SSH remote access (sudo raspi-config)

Scripts and Python code loaded in /home/pi/bin/AutoGro/

crontab.pi should be the pi user's crontab (to wrap logs)

rc.local should be copied to /etc (or the one there edited to start Python code at boot)

Make sure scripts have execute bit set chmod 755

AutoGro code runs as the pi user
________________________________

Bring Pi current with the following commands

sudo apt update
sudo apt full-upgrade
sudo apt clean
____________________________________

To use MCP3008 for AtoD install python lib with below:
sudo pip3 install adafruit-circuitpython-mcp3xxx
_____________________________________

To configure Pi static ip address

Edit /etc/dhcpd.conf
Look at examples
Wifi is wlan0, Ethernet is eth0
interface wlan0
static ip_address=
etc
